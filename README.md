# 체스 요구사항 분석

## 1. 입력

- [ ]  게임 여부를 입력 받는다.
    - [ ]  게임 시작은 start, 게임 종료는 end

## 2. 출력

- [ ]  안내 문구를 출력한다.
- [ ]  체스판을 출력한다.

## 3. 비즈니스

- [ ]  체스판을 생성
    - [ ]  8 X 8 크기를 갖는다.
    - [ ]  검은색, 하얀색 진영으로 나뉜다.
    - [ ]  각 진영은 폰(8개), 퀸(1개), 킹(1개), 나이트(2개), 비숍(2개), 루크(2개) 를 가진다.
- [ ]  말 이동
    - [ ]  말은 각각의 방법에 따라 이동한다.
    - [ ]  이동하려는 위치가 체스판을 넘을 수 없다.
    - [ ]  이동하려는 위치에 같은 진영의 말이 있을 경우 움직을 수 없다.
    - [ ]  나이트를 제외한 말은 다른 말을 넘어갈 수 없다.
